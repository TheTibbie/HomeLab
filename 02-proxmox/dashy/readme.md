# Dashy (Service Dashboard)

## Summary

Dashy provides a single landing page for accessing internal homelab services — hypervisor UIs, PBS, DNS, monitoring, and anything else that needs a persistent bookmark. It's a usability layer, not infrastructure, but it reduces friction during day-to-day operations and service validation across nodes.
